--Local'imizde repo oluşturabilmek için 
1-Çalışacağımız klasörü seçip ya da yeni bir klasör oluşturup, klasörün yolunu gösteren bölüme cmd yazıp enter tapıp açmış olduğumuz klasörün yolunda terminalimizi açmış oluruz.
2-Ve bu dizinde açılan terminale git init yazarız ve initialized empty git repository yazısını gördüğümüzde bu çalışma alanında boş bir repository oluşturmuş oluruz.
